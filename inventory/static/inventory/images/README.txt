Custom navbar logo
------------------
Replace warewolf-logo.svg with your own logo file.

- Use the same filename (warewolf-logo.svg) so no code changes are needed, or
- Use a PNG: save as warewolf-logo.png and in base.html change the img src to use warewolf-logo.png instead of .svg

Recommended size: about 36px height (width auto). SVG or PNG with transparent background works best on the dark navbar.
